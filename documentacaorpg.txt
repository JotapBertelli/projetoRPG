Backend:RPG desenvolvido utilizando apenas a linguagem de programação Python aplicando Laços de repetição (for, while e do while) e Condição (if/elif). 
Definição:
Laços de repetição: São estruturas que permitem executar um bloco de código repetidamente até que uma condição especificada seja satisfeita. 
Existem três tipos principais de laços em programação: for, while e do-while. O laço for é utilizado quando se sabe o número exato de vezes que a repetição deve ocorrer.

If/Elif: Permite que o programa execute diferentes blocos de código com base em condições específicas, podendo ser interpretadas como "Se" ou "Se não".
